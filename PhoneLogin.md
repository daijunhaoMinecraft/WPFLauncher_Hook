# 教程: 如何使用手机号登录
首先,你需要先下载此程序: [loginByPhone.7z](https://x19.fp.ps.netease.com/file/6958cabbe425efc35470f58bD1D9dwBW07) 然后解压<br/>
或者前往 [GitHub](https://github.com/bubbles-wow/WPFLauncher_Hook/blob/main/loginByPhone.exe) 上下载 <br/>
下载完成的文件如图所示(SHA3-256: d01ec882ef96cfb877ea50eca3d99ce68837190512da592195d0e35176009f55 ) <br/>
![PixPin_2026-01-03_15-54-24.png](https://x19.fp.ps.netease.com/file/6958cb3691474aac38f82a073TJRvaQe07) <br/>
双击打开后,按照步骤操作
![PixPin_2026-01-03_15-27-50.png](https://x19.fp.ps.netease.com/file/6958cb9aa236fe50514bb152oNEzEBqV07) <br/>
复制RESULT里面的值,格式如下: <br/>
{"sauth_json":"..."} <br/>
然后打开我的世界启动器,在**登录界面**点击右上角的"X" <br/>
![PixPin_2026-01-03_15-59-25.png](https://x19.fp.ps.netease.com/file/6958cc669acad9b51a831419IlGFT3iE07) <br/>
然后在弹出的对话框中选择"确定" <br/>
![PixPin_2026-01-03_16-00-55.png](https://x19.fp.ps.netease.com/file/6958cd8b08312b47e45d5f97HiabU4yy07) <br/>
然后在下方的输入栏中粘贴你刚刚复制的值,随后点击确定 <br/>
![PixPin_2026-01-03_16-03-42.png](https://x19.fp.ps.netease.com/file/6958cd8b577c4baa7dd95e74PqC8LyEb07) <br/>
之后你的手机号账号便登录上去了 <br/>
**重新登录的步骤和刚刚一样(到登录界面后点击"X"然后点击"确定",但是这次会自动保存RESULT的值,所以直接点击"确定"即可)** <br/>
